# Deployment Guide

## Production Deployment

### Prerequisites
- Railway Account (or similar PaaS)
- Supabase Project
- VAPI Account
- Google Maps API Key

### Vercel (Frontend)
1. Import project to Vercel
2. Configure Environment Variables (see `.env.example`)
3. Deploy

### Railway (Backend)
1. Connect GitHub repo
2. Set Environment Variables
3. Deploy
