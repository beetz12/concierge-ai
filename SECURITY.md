## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in Concierge AI, we encourage you to let us know right away. We will investigate all legitimate reports and do our best to quickly fix the problem.

Please report vulnerabilities by opening a **private security advisory** at:
https://github.com/NexAiAdvisors/concierge-ai/security/advisories/new

This ensures the vulnerability is reported privately and not exposed publicly until a fix is ready.
